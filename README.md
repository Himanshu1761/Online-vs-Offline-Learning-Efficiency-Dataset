# Online-vs-Offline-Learning-Efficiency-Dataset
This project focuses on analyzing the effectiveness of online and offline learning methods using Exploratory Data Analysis (EDA) and Machine Learning techniques.

The dataset examines how factors such as:

* Study Hours
* Focus Level
* Retention Score
* Learning Mode
* Subject Preference
* influence student academic performance and exam scores.

## The project also includes:

* Feature Importance Analysis
* Feature Selection
* Predictive Modeling
* Behavioral Pattern Analysis
* Multi-variable Educational Insights

# About Dataset
This dataset explores the effectiveness of online and offline learning methods by analyzing key factors such as:

Study Hours
Focus Level
Retention Score
Exam Performance

It is designed to simulate real-world learning scenarios and helps in understanding how different study environments influence student outcomes.

## Key Analytical Questions Explored
Some major questions analyzed in this project include:

Does study hours influence exam performance?
Is focus level associated with better grades?
Do online learners require more study hours for similar performance?
Which subject benefits most from higher focus levels?
Are lower-grade students concentrated in specific behavioral patterns?
Which combination of learning mode, focus level, and study hours produces the highest probability of A1 grades?
Does retention level influence academic performance?

📈 Key Insights
✔️ Focus Level shows strong association with grades
Students with very high focus levels demonstrated significantly higher concentration in A1 and A2 grade categories.
✔️ Study Hours positively influence performance
Higher study-hour commitment generally showed upward movement toward better grades.
✔️ Offline learning mode performed better in top-grade concentration

Feature importance techniques were used to identify the most influential variables affecting exam scores.

## Machine Learning Models

Model	R² Score	MAE

## Model Interpretation
Linear Regression achieved the best performance, indicating that:

The relationship between study behavior and exam performance is largely linear.
Behavioral and learning variables strongly explain academic outcomes.

🎯 Final Conclusion
This project demonstrates how educational behavior patterns such as:

Focus
Study duration
Learning environment
Retention capability
can significantly influence academic performance.

The analysis highlights that:
Focus level plays a stronger role than study duration alone.
Offline learning combined with high focus and structured study hours is associated with stronger academic outcomes.
